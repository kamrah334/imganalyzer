# SEO Image Analyzer

This tool analyzes images, generates professional SEO-optimized titles and keywords, and exports results to a CSV file.

## Features
- Upload multiple images
- AI-generated captions (BLIP model)
- DeepAI-powered SEO title enhancement & keyword generation
- CSV download (Filename, Title, Keywords)
- Safe content filtering

## Setup

1. Clone this repo:
```bash
git clone https://github.com/YOUR_USERNAME/SEO_Image_Analyzer.git
cd SEO_Image_Analyzer
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set your DeepAI API Key:
```bash
export DEEPAI_API_KEY="your_api_key_here"
```

4. Run the app:
```bash
python app.py
```

## Deployment
- Can be deployed to Hugging Face Spaces or Vercel (Python backend).
